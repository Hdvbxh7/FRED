Cette classe encapsule les opérations Git (clone, pull, commit, push) en utilisant la bibliothèque JGit. Elle gère l'authentification, le clonage initial des dépôts étudiants, la mise à jour via pull, et la publication des résultats d'évaluation via commit et push, tout en gérant les erreurs et les conflits potentiels.
