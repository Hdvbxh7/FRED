Cette classe abstraite hérite d'[[Evaluateur.java|Evaluateur]] et sert de base pour tous les évaluateurs de compilation, comme [[EvaluateurCompilationJava.java|EvaluateurCompilationJava]] et [[EvaluateurCompilationAda.java|EvaluateurCompilationAda]]. Elle maintient une liste de fichiers à compiler et définit l'interface commune pour vérifier si du code source peut compiler correctement.
